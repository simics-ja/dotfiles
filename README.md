# My dotfiles

## Usage

```bash
git clone https://github.com/simics-ja/dotfiles.git
cd dotfiles
sh setup.sh
```

## Option

### local config

**WSL example:**

```bash
cp git/config/local.wsl.sample git/config/local
cp zsh/.zshrc.local.wsl.sample zsh/.zshrc.local
```
